2️⃣ FULL AI FORECAST MATHEMATICAL MODEL

We move from rule-based to weighted probability model.

📊 Feature Vector

For each product:

X = [
  FX_volatility,
  vessel_count,
  refinery_output_index,
  depot_spread,
  regulation_impact,
  demand_index,
  historical_volatility,
  trader_sentiment_score
]
🎯 Logistic Probability Model
P(increase) = 1 / (1 + e^-(W·X + b))

Where:

W = weight vector

b = bias

X = feature input

Confidence Score
confidence = max(P(increase), P(drop), P(stable))
Expected Range Calculation
volatility = historical_std_dev(product)

expected_min = current_price - (volatility * risk_factor)
expected_max = current_price + (volatility * risk_factor)
Risk Factor Formula
risk_factor =
  1
  + (FX_volatility * 0.2)
  + (regulation_impact * 0.3)
  + (refinery_disruption * 0.4)
